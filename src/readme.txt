document numbering must be as follows:
"COILCHAIN_yynnnTYPE_NAME.tex"
yy the year of the document
nnn the document number
TYPE the document type
NAME a short name to identify the content

the numbering of invoice-offer-letter is independent but follows the same principle:
"COILCHAIN_yymmddTYPE_DESTINARY_SUBJECT.tex"
yymmdd the date of edition
TYPE LTR/OFR/INV
RECIPIENT a short description of the recipient
SUBJECT a short subject
